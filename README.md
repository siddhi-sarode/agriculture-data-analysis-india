 🌾 Agriculture Data Analysis Project

📌 Overview

This project analyzes agricultural crop production data in India to discover trends, top-performing crops, and production patterns over time.



 📊 Key Insights

* Coconut is the highest produced crop in India
* Sugarcane, Rice, and Wheat are major contributors
* Agricultural production varies significantly across states
* Overall production shows an increasing trend over the years
* Sudden drops may indicate missing or incomplete data



 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook (VS Code)



 📂 Dataset

The dataset used in this project is publicly available on Kaggle:

🔗 https://www.kaggle.com/datasets/iamtapendu/crop-production-data-india
